# Cockatoo

- [x] setup [Tensorflow](https://www.tensorflow.org/)
- [x] ingest video feed
- [ ] detect cockatoos
- [ ] ping [HomeAssistant](https://www.home-assistant.io/)

To run:
1. Add a .env file with the below filled in
  
```
PORT=
CAMERA_HOST=
CAMERA_PORT=
CAMERA_USER=
CAMERA_PASSWORD=
```

2. run `npm run start`
